# Agentic Mobile

React Native mobile app for Agentic Web3 platform.

## Setup

```bash
cd mobile
npm install
npm start
```

## Features

- Cross-platform (iOS, Android)
- Neo Glow themed UI
- Shared API client with web app
- Real-time blockchain data
- AI agent integration

## Development

- Use Expo for development and testing
- Supports both iOS and Android
- Can run on physical devices or simulators
